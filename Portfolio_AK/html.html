Html og Css

Tilbage til 1. semester?

I løbet af den første til på 1.semester blev jeg introduceret til html og hermed css. html er et kode program som lægger en grundstruktur til en hjemmeside. Se øverste billede.

I forbindelse med html har jeg så lært css. Css er stadig kodning men denne gang på designet, farver, størrelser, placering osv. se billedet til venste.

Sådan ser det så ud med koden bag coverbilledet på min forside. Det kræver en masse kodning for noget så enkelt. Det sidste jeg skal kunne i denne forbindelse er Javascript som gør de mest avanceret ting som hverken html eller css kan.
